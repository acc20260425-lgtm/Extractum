# Documentation Archive

This directory stores reference material that is no longer part of the active
documentation flow.

- `migrations-pre-baseline-reset/`: historical SQL and runner-managed Rust
  migration code from before the current-schema baseline reset.
- `database-schema-legacy-analysis.md`: historical schema debt analysis kept
  for background and rationale.

Archive files can explain how the project got here, but current behavior should
be documented in the root `docs/*.md` files.
